# Paper Materials Repository

This repository provides the codes, figures, and data associated with the paper "A teleconnection pathway linking cross-basin extremes triggered the record-breaking 2024 marine heatwave in the East China Seas" DOI: https://doi.org/10.1088/1748-9326/ae5314.

## 📂 Repository Structure

- **Code/**  
  Contains all the scripts used for data processing and figure plotting in the paper.

- **Fig/**  
  Contains the figures shown in the paper.

- **Data**  
  Model data of the paper is available at: 10.6084/m9.figshare.31820290. A brief description is provided here (ModelDataList.txt).

## 📊 Data Availability

All datasets used in this study are publicly available or provided herein. 
They can be accessed via the corresponding links given in the paper and in this section.
